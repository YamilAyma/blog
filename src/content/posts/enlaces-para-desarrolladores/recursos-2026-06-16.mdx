Librerias que pueden ser útiles si estás trabajando en frontend y UI.

---

Shadcn Phone Input: Componente de entrada de teléfono simple y formateado para Next.js, creado con shadcn/ui y libphonenumber-js.
Link: https://shadcn-phone-input-five.vercel.app/

gooey-toast: Componente de pan tostado pegajoso y transformable para React, construido sobre Sonner con animaciones Framer Motion.
Link: https://goey-toast.vercel.app/

ScrollX-UI: Una colección de código abierto de componentes animados, interactivos y totalmente personalizables para crear interfaces de usuario impactantes y memorables.
Link: https://github.com/Adityakishore0/ScrollX-UI

Kutty: Kutty es un plugin de Tailwind para la creación de aplicaciones web. Cuenta con un conjunto de componentes accesibles y reutilizables que se utilizan habitualmente en aplicaciones web.
Link: https://kutty.netlify.app/
