4 recursos que encontré para ti respecto a backend y servicios.

---

Ghost: Plataforma moderna de código abierto para crear blogs y boletines de noticias como alternativa a Wordpress.
Link: https://github.com/TryGhost/Ghost

Trilium Notes: Aplicación gratuita, de código abierto y multiplataforma para tomar notas de forma jerárquica, centrada en la creación de grandes bases de conocimiento personales.
Link: https://triliumnotes.org/

memos: Herramienta de toma de notas de código abierto y autoalojada, diseñada para capturar información rápidamente. Compatible con Markdown, ligera y totalmente personalizable.
Link: https://usememos.com/

docker: La plataforma de contenedores estándar para desarrollar, enviar y ejecutar aplicaciones en cualquier lugar.
Link: https://www.docker.com/
