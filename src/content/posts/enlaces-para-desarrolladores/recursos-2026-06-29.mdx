Siguiendo la serie de recursos frontend, aquí hay unos más si te interesan o deseas probar nuevas librerías para tus proyectos.

---

WarcraftCN: Un conjunto de componentes inspirados en la estética clásica de la interfaz de usuario de Warcraft III RTS. Código abierto, listo para copiar y pegar. Creado por fans. Sin afiliación.
Link: https://www.warcraftcn.com/

Retroui: Una librería de interfaz de usuario para React + TailwindCSS con estilo neobrutalista, ideal para crear aplicaciones web llamativas.
Link: https://www.retroui.dev/

Oat: Biblioteca de componentes de interfaz de usuario semántica, ultraligera, basada en HTML y CSS, sin dependencias. 
Link: https://github.com/knadh/oat

BetterAuthUI: Conjunto de componentes de interfaz listos para implementar sistemas de autenticación modernos.
Link: https://better-auth-ui.com/
