Bienvenida semana, aquí 3 proyectos que pueden ser de curiosidad si estás trabajando en frontend como infraestructura.

---

Logto: Plataforma de gestión de identidad (IAM) de código abierto para añadir autenticación y autorización profesional a aplicaciones SaaS.
Link: https://logto.io/

Soundcn: Libreria de codigo abierto con +700 efectos de sonido para apps webs, instalable de forma similar a como lo harías con Shadcn.
Link: https://github.com/kapishdima/soundcn

Park UI: Sistema de diseño moderno basado en Panda CSS y Ark UI para componentes de alta calidad.
Link: https://park-ui.com/
