4 recursos que encontré para ti respecto a desarrollo web.

---

simpleParallax.js: Librería de JavaScript ultra ligera y fácil de usar para añadir efectos de paralaje a cualquier imagen.
Link: https://simpleparallax.com/

Video Editor by Kimu: Editor de video ligero basado en web que permite recortes y ediciones rápidas mediante ffmpeg.wasm.
Link: https://github.com/trykimu/videoeditor

Heerich.js: Motor minimalista para renderizar escenas 3D de vóxeles utilizando gráficos vectoriales SVG.
Link: https://meodai.github.io/heerich/

Hyperframes: Framework de HeyGen para generar videos usando HTML nativo y atributos de datos.
Link: https://github.com/heygen-com/hyperframes
