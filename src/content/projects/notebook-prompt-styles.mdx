## Descripción

Un repositorio en [GitHub](https://github.com) que recopila y organiza prompts para personalizar las diapositivas generadas en [NotebookLM](https://notebooklm.google.com), una herramienta de [Google](https://google.com) para el estudio y la investigación. El proyecto incluye estilos prediseñados en formato [YAML](https://yaml.org) listos para copiar y pegar.
